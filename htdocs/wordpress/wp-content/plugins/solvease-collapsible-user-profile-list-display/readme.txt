This plugin helps to make user/ subscriber for making their profile and display into post with shortcode.

== Description ==

This plugin helps to make user/ subscriber for making their profile with Biographical Info, Profile Image, Social Media Profile links, Occupation ,Company Name, Education and all the subscriber's profile will display into post or page with shortcode [subscribers].

== Installation ==

1. Upload `collapsible_user_profile_display` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make user profile from admin panel.
4. Place Shortcode `[subscribers]` in your page or post.

== Frequently asked questions ==

= How users will make Profile? =

Ans: Every subscriber can login into admin panel using email and password. After login they will see an admin panel. By providing all information into the required fiels they can complete the profile.
